# 🏥 Medical Insurance Cost Prediction

## 📌 Project Overview
The **Medical Insurance Cost Prediction System** is a machine learning project that predicts an individual's annual medical expenses based on personal and lifestyle factors such as age, BMI, smoking habits, stress level, and gender.

This project demonstrates how data analysis and machine learning can be used to estimate healthcare costs and identify key contributing factors.

---

## 💡 Project Idea
The goal of this project is to:
- Analyze medical insurance data
- Identify factors affecting medical costs
- Build a predictive model using machine learning

---

## 🎯 Use Cases
- 🏢 Insurance companies for premium calculation  
- 📊 Healthcare analytics  
- 💰 Personal financial planning  
- 🏛️ Policy making and risk assessment  

---

## ⚙️ Approach

### 1. Data Collection
Dataset includes:
- Age  
- Gender  
- BMI  
- Smoking status  
- Stress level  
- Annual medical cost (target)

### 2. Data Preprocessing
- Handling missing values  
- Encoding categorical variables  
- Feature selection  

### 3. Exploratory Data Analysis (EDA)
- Histograms  
- Scatter plots  
- Box plots  
- Heatmaps  

### 4. Model Training
- Train-test split  
- Models used:
  - Linear Regression  
  - Random Forest Regressor  

### 5. Model Evaluation
- Mean Absolute Error (MAE)  
- Mean Squared Error (MSE)  
- R² Score  

---

## 🧠 Solution

### 🔹 Linear Regression
- Simple and interpretable  
- Works well for linear relationships  

### 🔹 Random Forest Regressor
- Handles non-linear data  
- Better accuracy  
- Reduces overfitting
